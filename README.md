# Тихий час

Красивый настольный таймер для macOS и Windows. Можно задать часы, минуты и секунды, поставить отсчёт на паузу и выбрать быстрый пресет. Когда время заканчивается, приложение показывает большое уведомление и подаёт звуковой сигнал.

## Запуск для разработки

Установите [Node.js](https://nodejs.org/) версии 20 или новее, затем выполните:

```bash
npm install
npm run desktop
```

## Сборка приложения

На macOS:

```bash
npm run build:mac
```

На Windows:

```bash
npm run build:win
```

Готовые файлы появятся в папке `release`.

## Готовая Windows-версия через GitHub

После каждого обновления ветки `main` GitHub автоматически собирает Windows-версию:

1. Откройте вкладку **Actions** в репозитории.
2. Выберите последний успешный запуск **Build Windows app**.
3. Внизу страницы скачайте архив **windows-app**.
4. Распакуйте архив и запустите установщик `.exe` либо portable-версию.

Windows может показать предупреждение SmartScreen, потому что приложение не подписано платным сертификатом разработчика. Для личного использования выберите **Подробнее → Выполнить в любом случае**.
